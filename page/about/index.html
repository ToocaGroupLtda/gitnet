About

gitnet is developed by NetLab at UW

gitnet is a Python 3 package for data mining source code repositories and mailing lists. It is intended to produce datasets that can be used in quantitative and social networks research.

gitnet is developed by members of the NetLab team at University of Waterloo:

  • Jillian Anderson
  • Joel Becker
  • Steve McColl
  • Dr. John McLevey (PI)
